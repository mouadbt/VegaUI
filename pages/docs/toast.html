Toast

A non-blocking notification component. Uses the card component for content and data-dismiss to cancel.

Default

With Dismiss

Placements

Persistent & Clear

Element-based

Usage

import { toast, toastEl, toastClear } from './index.js';

toast('<div class="card" style="padding: 1rem 1.25rem;">Settings saved.</div>');

toast(`...html...`, { placement: 'bottom-center', duration: 5000 });

toastEl(document.getElementById('my-template'));

toastClear();
